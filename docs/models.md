# Models

## Current Live Demo Weights

The live demo currently supports these selectable weights:

- `best_multiple.pt`
- `best_single.pt`
- `m11best.pt`
- `m8best.pt`
- `s11best.pt`
- `s8best.pt`

The default is `best_multiple.pt`.

The live demo expects these `.pt` files to be available in `rgbd/live/models/`.
