# UI Controls

## Data Collection UI

### Buttons

- `Capture Frame (Enter)` - freeze the current frame
- `Retake Frame (R)` - discard the current capture and return to live mode
- `COMMIT DATASET SNAPSHOT (S)` - save the current package
- `Preview Point Cloud (P)` - open the point cloud preview

### Label Selection

- `Class 00: Copper Mass Asset`
- `Class 01: Steel Structural Scrap`

### Keyboard Shortcuts

- `Enter` - capture
- `S` - save
- `R` - retake
- `P` - point-cloud preview
- `Q` - quit

## Live Demo UI

### Top Bar

- FPS display
- detection count
- current model name
- model mode indicator
- model button
- info button

### Mouse Controls

- click `Model` to open the six-model selector
- click `Info` to open or close the model information panel

### Keyboard Shortcuts

- `S` - save a snapshot from the demo
- `R` - start or stop recording
- `Q` - quit the demo
